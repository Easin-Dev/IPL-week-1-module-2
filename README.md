
# Week 1 - Module 2: Operators & Conditionals

## Topics Covered

1. **Arithmetic Operators**
    - Basic mathematical operations (+, -, *, /, %, **)
    - Order of operations and precedence

2. **Relational Operators**
    - Comparison operators (==, !=, <, >, <=, >=)
    - Evaluating relationships between values

3. **Logical Operators**
    - AND, OR, NOT operations
    - Boolean logic and truth tables
    - Combining conditions

4. **Conditional Statements**
    - if, else, elif structures
    - Control flow decision-making
    - Nested conditions

## Key Concepts

- Understanding operator precedence
- Writing complex logical expressions
- Implementing branching logic in programs

## Practice

Apply these concepts to solve conditional problems and implement decision-based programming logic.
